# Deployment Guide

Recommended deployment:

- Frontend: Vercel
- Database: Neon, Supabase, Railway, or PlanetScale
- File storage: Cloudinary or S3-compatible storage
- Email: Gmail SMTP, Resend, or Mailgun

## Production Checklist

- Add all environment variables
- Run database migration
- Test all API routes
- Add screenshots to README
- Add live demo link
- Pin this repository on GitHub profile
